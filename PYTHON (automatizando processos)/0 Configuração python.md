# Instalando virtual env 
1. abre o cmd
```shell
# na pasta do projeto
>virtualenv env
```
2. Entra na pasta script da pasta env
```shell
> cd env
> cd scripts
> active
```
### instale na pasta env a biblioteca xlrd,sellenium e request
```
pip install xlrd
pip install selenium
pip install requests
```
