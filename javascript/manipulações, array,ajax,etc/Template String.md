- cria string forma mais limpa.

<span style="color:red">Modo Antigo</span> 
```js
let nome = `Bonieky`;
let idade = 90;

let frase = 'Meu nome é'+nome+  'e eu tenho'+idade+ 'anos';
```

# <span style="color: #00FF00">Modo Moderno</span> 
```js
let nome = `Bonieky`;
let idade = 90;

let frase = `Meu nome é ${nome} e eu tenho ${idade};`
```
