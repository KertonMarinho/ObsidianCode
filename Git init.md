```shell
cd <pasta do projeto>
```
``VC PODE COPIAR O NOME DA PASTA DO PROJETO DO EXPLORER E COLAR COM CD``

2. De o primeiro git add no projeto:

```shell
git add*
```
3. faça o commit:

```shell
git -m "Commit inicial"
```
