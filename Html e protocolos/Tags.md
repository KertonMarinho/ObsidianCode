# Tag link
```html
<a href="Endereço site">texto</a>

Endereço do site - valor do atributo
```
# Imagens

```html
<img src="endereço da imagem"/>
src - surce fonte da imagem
tag Endtag
```
- Ajustar o tamanho da imagem

```html
<img src ="endereço da imagem"  width="100px" height="100px"/>
```
